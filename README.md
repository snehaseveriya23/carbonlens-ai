# CarbonLens AI

### Measure Today. Reduce Tomorrow. Shape a Greener Future.

CarbonLens AI is an intelligent carbon footprint awareness platform that helps individuals understand, track, and reduce their environmental impact through personalized insights, carbon footprint analysis, and sustainability recommendations.

## Features

* Carbon Footprint Calculator
* Emission Breakdown Dashboard
* Eco Score
* Carbon Hotspot Detection
* Personalized Sustainability Recommendations
* Carbon Reduction Simulator
* Weekly Green Challenge Planner

## Tech Stack

* React
* Vite
* JavaScript
* Chart.js
* CSS
